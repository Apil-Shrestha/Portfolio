Apil Shrestha — Developer & Designer Portfolio
A collection of frontend projects built with pure HTML, CSS, and Vanilla JavaScript — no frameworks, no dependencies. Each project was designed and coded from scratch with a focus on clean UI, smooth interactions, and real functionality.

🌐 Personal Portfolio
A fully responsive personal portfolio featuring a dark/light mode toggle, animated typing effect, scroll-triggered fade-in animations, sticky navbar with frosted glass effect, and a working contact form. Built with SEO meta tags, Open Graph support, and mobile-first design.
🛒 KURA Store — E-Commerce Website
A responsive e-commerce frontend with 20+ products across 5 categories. Includes live search, category filters, a price range slider, sort options, a slide-out shopping cart with quantity controls, product wishlist, modal popups, and a full checkout form with card formatting. Zero external libraries.
✅ Focus — Task Manager App
A Kanban-style productivity app with a 3-column board (To Do → In Progress → Done). Features drag-and-drop cards, priority levels (High / Medium / Low), due date tracking with overdue detection, custom tags, keyword search, and filter by priority. All tasks persist across sessions via localStorage.
📚 StudyNest — Study Group Platform
A multi-page study platform with personalised onboarding — users enter their name on first visit and the app remembers them. Includes a 7-day schedule board, a shared resource library, a notes editor with formatting tools, and live session management. All data saved locally in the browser.
🌤️ SkyLens — Weather App
A weather dashboard with city search, 24-hour hourly forecast scroll, 7-day outlook, UV index, air quality index, sunrise/sunset times, and wind/humidity/pressure stats. Features animated weather icons and supports 7 pre-loaded cities with realistic weather simulation.

Tech Stack: HTML5 · CSS3 · Vanilla JavaScript · localStorage API · CSS Animations · IntersectionObserver API
